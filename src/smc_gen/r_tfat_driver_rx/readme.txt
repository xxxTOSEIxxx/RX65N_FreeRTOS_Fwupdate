r_tfat_driver_rx
=========


Overview
--------
This memory driver interface connects TFAT and Device Drivers(USB etc).
TFAT is a FAT file system which just requres small memory.
TFAT is based on FatFs.
<br/>
Please refer to the
 https://www.renesas.com/mw/tfat


Features
--------
* This memory driver interface connects TFAT and Device Drivers(USB etc).
